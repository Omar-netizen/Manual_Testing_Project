# OpenCart E-Commerce – Manual Testing Project

End-to-end manual testing of the [OpenCart demo application](https://demo.opencart.com/), covering core e-commerce workflows.

---

## 📋 What's Included

| Sheet | Description |
|---|---|
| **Test Cases** | 22 test cases across 7 modules |
| **Bug Report** | 3 bugs logged with severity & steps to reproduce |
| **Test Summary Report** | Module-wise pass/fail metrics (~95% pass rate) |

---

## 🧪 Modules Tested

- User Registration & Login
- Search
- Shopping Cart
- Checkout
- Product Page
- Navigation & UI

---

## 🛠️ Techniques Used

- Software Testing Life Cycle (STLC)
- Functional, Negative & UI Testing
- Test Case Design & Bug Reporting
- Severity & Priority Classification
